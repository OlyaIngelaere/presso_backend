# ☕ Presso - Coffee Capsule Finder

This project contains the backend for the Presso app (https://github.com/r0879042/presso-app).

---

## 🛠 Tech Stack

- Framework: Laravel (PHP)
- Database: SQLite
- Deployment: Docker / [Render](https://render.com/)
